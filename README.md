# Lock Prototyping raspberrypi

## Require

node

## インストール

```
$ sudo yum install bluez bluez-libs bluez-libs-devel
$ npm install
```

## BLEペリフェラルの起動

```
$ sudo /home/pi/.nodebrew/current/bin/node proto.js
```
